1. Интерфейс:
    - Главная страница - карта (с точками, зонами-кластерами);
    - Панель с адресами из точек;
    - Личный кабинет;
    - Панель с аналитикой;
    - Панель управления.

2. Функционал:
    - Оптимальная кластеризация точек;
    - Автоматическая группировка заказов;
    - Распределение зон по курьерам;
    - Ручная корректировка;
    - Система аккаунтов;
    - Интерактивная карта;
    - Управление точками, заказами, кластерами, аккаунтами;
    - Загрузка/Выгрузка заказов из .xls.

3. Карта:
    - Интеграция api яндекс.карт;
    - Отображение точек и зон доставки.

4. Управление:
    - Горячие клавиши;
    - Работа с картой при помощи мыши.
