1. Интерфейс:
    - Главная страница - карта (с точками, зонами-кластерами, путями);
    - Личный кабинет;
    - Панель с важной информацией и выходом на смену;
    - Панель управления.

2. Функционал:
    - Оптимальная кластеризация точек;
    - Автоматическая группировка заказов;
    - Распределение зон по курьерам;
    - Ручная корректировка;
    - Выход на смену;
    - Система аккаунтов;
    - Интерактивная карта;
    - Управление точками, заказами, кластерами, аккаунтами, сменами.

3. Карта:
    - Интеграция api яндекс.карт;
    - Отображение точек и зон доставки.

4. Управление:
    - Горячие клавиши;
    - Работа с картой при помощи мыши.
