pathclustering (рабочее название)

Планируется создать сайт, целью которого будет оптимизировать и повысить эффективность работы служб доставки.
Основной задачей будет создание системы, способной автоматизировать процесс распределения заказов между курьерами.
Для решения будет внедрена система кластеризации точек. Она позволит автоматически сгруппировать заказы по путям, формируя оптимальные зоны.
Распределение задач между курьерами минимизирует время на перемещение и обеспечит максимально быструю доставку.
Также будет разработана система аккаунтов, где рабочие смогут "выйти в смену" и получить зону заказов, в которой надо развести, а админы смогут иметь контроль над всем в случае чего.
С помощью библиотек и api добавим функциональную карту, по которой можно будет перемещаться. На ней будут отмечены точки и зоны-кластеры.
Планируется предоставить возможность загрузить/выгрузить заказы/точки в/из Excel файла.
Весь функционал сайта будет предоставлен и в телеграм-боте (если нам позволят).
